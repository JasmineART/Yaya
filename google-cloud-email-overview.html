🚀 Quick Start (5 Steps)

1 Run Setup Script

cd /workspaces/Yaya
./setup-google-email.sh

2 Follow OAuth Setup Prompts

The script will guide you through creating OAuth credentials and storing them securely.

3 Deploy Cloud Function

Script automatically deploys your function and gives you the URL.

4 Test Integration

# Test newsletter signup
curl -X POST YOUR_FUNCTION_URL \
  -H 'Content-Type: application/json' \
  -d '{"type":"newsletter","data":{"email":"test@example.com"}}'

5 Check Email

You should receive a test email at faeriepoetics@gmail.com
