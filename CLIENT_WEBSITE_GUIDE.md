## 🛠️ Troubleshooting

### Common Issues & Solutions

**1. Can't Login to Admin Panel**
```
Problem: Login page shows "Invalid credentials"
Solution: 
- Ensure you're using: Username: admin, Password: yaya2025!
- Check for typos (password is case-sensitive)
- Try refreshing the page
- If still failing, contact support
```

**2. Changes Not Appearing on Website**
```
Problem: Made changes in admin but website looks the same
Solution:
- Wait 30 seconds for changes to propagate
- Refresh the website page (Ctrl+F5 or Cmd+Shift+R)
- Clear browser cache if needed
- Check that you clicked "Save" or "Update" button
```

**3. Undo Button Not Working**
```
Problem: Undo button is grayed out or not responding
Solution:
- Undo button is only active after making changes
- Make sure you've performed an action first
- If no history available, button will be disabled
- Try refreshing admin page if persistent
```

**4. Images Not Loading**
```
Problem: Product images show as broken links
Solution:
- Ensure image URL is complete and starts with https://
- Test image URL by pasting in browser address bar
- Use supported formats: .jpg, .png, .gif
- Check image file isn't too large (recommended under 2MB)
```

**5. Coupon Codes Not Working**
```
Problem: Customers report coupon codes invalid
Solution:
- Check coupon code spelling in admin panel
- Ensure coupon is marked as "Active"
- Verify discount type and value are correct
- Test the coupon yourself on checkout page
```

### When to Contact Support
- Admin panel completely inaccessible
- Website showing error messages
- Payment processing issues
- Email system not sending messages
- Need help with advanced customizations
