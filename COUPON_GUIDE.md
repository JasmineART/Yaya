## ⚠️ Important Notes

### Code Restrictions
- ❌ Only ONE code per order
- ❌ Codes cannot be stacked
- ❌ PASTEL requires minimum 2 items
- ❌ Codes are case-insensitive (pastel = PASTEL)

### Discount Application
- ✅ Applied to subtotal BEFORE shipping
- ✅ Applied to subtotal BEFORE tax
- ✅ Persists through checkout
- ✅ Shows in cart and order summary

### Calculation Order
```
1. Subtotal (item prices × quantities)
2. - Discount (PASTEL/SUNCATCHER/WHIMSY)
3. = Discounted Subtotal
4. + Shipping ($9.99)
5. + Tax (8.5% of discounted subtotal)
6. = FINAL TOTAL
```

## 🛠️ How to Apply a Code

1. **Add items to cart** → Shop → Add to Cart
2. **Go to cart page** → Click "Cart" in navigation
3. **Enter code** → Type in discount code field
4. **Click "Apply Code"** → Green success message appears
5. **See discount** → Discount line shows in cart
6. **Proceed to checkout** → Discount carries through
